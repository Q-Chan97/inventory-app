# inventory-app

This is a mini message board project created as part of The Odin Project's NodeJS course. 

This project is meant to showcase my understanding of:
   - The Express framework
   - CRUD (Create, Read, Update, Destroy) operations
   - Multi Page applications
   - Relational databases and SQL

The technologies I used for this project are: 
   - Express
   - EJS
   - PostgreSQL
   - Express Validator

This project features:
   - An inventory management app for tracking spirit collections
   - CRUD functionality
   - Dynamic routing with parameterized URLs
   - Form handling with custom errors
   - Relational databases with foreign key reference
   - Static asset serving
   - 404 error handling
